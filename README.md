# JavaScript를 활용한 영단어 카드게임 - 공차팀
<br>

## 프로젝트 소개
* 영단어 카드 게임입니다. 단어가 주어지면 해당 단어로 문법을 고려하여 문장을 완성합니다.
* 문장의 결과에 따라 정답 또는 오답을 출력합니다.
